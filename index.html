Información Extra - Tipos de Botones en Bootstrap 3

Bootstrap 3 ofrece varias clases para estilos de botones:

  • btn-default: Botón neutro (gris).
  • btn-primary: Botón principal (azul).
  • btn-success: Éxito o confirmación (verde).
  • btn-info: Información (celeste).
  • btn-warning: Advertencia (naranja).
  • btn-danger: Peligro o acción destructiva (rojo).
  • btn-link: Estilo de enlace, sin fondo ni borde.

También se pueden usar modificadores de tamaño:

  • btn-lg: Botón grande
  • btn-sm: Botón pequeño
  • btn-xs: Botón extra pequeño
  • btn-block: Botón de ancho completo

Ejemplos:

<button class="btn btn-default">Default</button>
<button class="btn btn-primary">Primary</button>
<button class="btn btn-success">Success</button>
<button class="btn btn-info">Info</button>
<button class="btn btn-warning">Warning</button>
<button class="btn btn-danger">Danger</button>
<button class="btn btn-link">Link</button>

<button class="btn btn-primary btn-lg">Grande</button>
<button class="btn btn-primary btn-sm">Pequeño</button>
<button class="btn btn-primary btn-xs">Extra pequeño</button>
<button class="btn btn-primary btn-block">Botón de bloque</button>

Módulo 3 - Tablas

Las tablas de Bootstrap incluyen estilos como rayado, bordeado y hover. También puedes usar clases para encabezados.

<table class="table table-striped table-bordered table-hover">
  <thead><tr><th>Nombre</th><th>Edad</th><th>Correo</th></tr></thead>

    <tr><td>Ana</td><td>25</td><td>ana@mail.com</td></tr>
    <tr><td>Luis</td><td>30</td><td>luis@mail.com</td></tr>
  </tbody>
</table>

Módulo 4 - NavBar

La barra de navegación (NavBar) de Bootstrap sirve para crear menús de navegación responsivos y organizados. Puedes incluir enlaces, dropdowns, formularios de búsqueda, logotipos y controles de usuario.

  • .navbar: Clase base del contenedor de navegación.
  • .navbar-default: Estilo predeterminado (puede ser .navbar-inverse para fondo oscuro).
  • .navbar-brand: Se usa para el logotipo o nombre del sitio.
  • .nav + .navbar-nav: Contenedor para los elementos de navegación.
  • .navbar-right: Alinea elementos a la derecha.
  • .navbar-form: Para formularios de búsqueda u otros.
  • .dropdown: Menús desplegables dentro del navbar.
<nav class="navbar navbar-default">
    <div class="container-fluid">
    <div class="navbar-header">
    <a class="navbar-brand" href="#">Mi Sitio</a>
    </div>
    <ul class="nav navbar-nav">
    <li class="active"><a href="#">Inicio</a></li>
    <li><a href="#">Nosotros</a></li>
    <li class="dropdown">
    <a class="dropdown-toggle" data-toggle="dropdown" href="#">
    Servicios <span class="caret"></span>
    </a>
    <ul class="dropdown-menu">
    <li><a href="#">Web</a></li>
    <li><a href="#">Móvil</a></li>
    </ul>
    </li>
    </ul>
    <form class="navbar-form navbar-right" role="search">
    <div class="form-group">
    <input type="text" class="form-control" placeholder="Buscar">
    </div>
    <button type="submit" class="btn btn-default">Buscar</button>
    </form>
    </div>
    </nav>

Módulo 5 - Modal
